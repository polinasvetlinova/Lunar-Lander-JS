Lunar lander es un juego donde el objetivo para ganar es aterrizar la nave a una velocidad inferior a 10m/s, si se hiciera a una velocidad superior la nave explotara. Si se ganara en la pantalla aparecera nuestro superheroe para decir que hemos ganado con un mensaje de vicoria.

El juego ha sido desarrollado por Polina Svetlinova Petrova del curso 1DAM. En el proceso de creación ha sido de gran ayuda el esqueleto del profesor Miguel Ángel Urbina que nos facilito el codigo HTML, JS y CSS basicos para empezar. Han sido consultadas las siguientes paginas web para ayudarme a finalizar el proyecto: w3schools y Stack0verflow.
